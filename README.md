# document-metadata-service
[![.NET CI](https://github.com/Landers99/document-metadata-service/actions/workflows/ci.yml/badge.svg?branch=main)](https://github.com/Landers99/document-metadata-service/actions/workflows/ci.yml)

Document Metadata Service is a secure REST API for managing document metadata records such as title, description, file name, content type, file size, and tags. Users authenticate with JWT and can only manage their own records. The service is containerized for local development, tested through integration tests, and validated by CI.
